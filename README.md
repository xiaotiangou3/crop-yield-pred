# Crop Yield Prediction Model

## Description
This is an ML model trained on over 1,600 rows of satellite and climate data, including NDVI and soil moisture indices. This model is served via a **FastAPI** backend hosted on **Render**, providing real-time yield predictions.

## Try it out!

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone

# Step 2: Navigate to the project directory.
cd 

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Deployment URL
This model is deployed [here](https://crop-yield-pred.onrender.com) on Render.
